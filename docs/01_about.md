## 1 - Название проекта

**Splatera** - Интеллектуальный менеджер графических ресурсов.

## 2 - Описание

Это десктопное приложение для организации локальной библиотеки референсов, текстур, ассетов и текстовых сниппетов. Приложение не перемещает файлы, а создает виртуальную библиотеку ссылок, позволяя пользователю просматривать контент в удобной визуальной сетке с фильтрованием по автоматически и вручную добавленным тегам и по преобладающим (в случае изображений) цветам.
## 3 - Пользователь

Основная аудитория: 
- Графические дизайнеры
- 3D-художники
- UI/UX специалисты.
- Frontend-разработчики

**Зачем:** Чтобы навести порядок в хаосе папок «Downloads» и «New Folder (2)», быстро находить нужные визуальные образы для мудбордов и иметь под рукой локальный аналог Pinterest без привязки к интернету.

## 4 - Ключевые функции (MVP)

1. Автоматическая привязка тега к элементу по его расширению, вместе с ручным добавлением тегов.
2. Отображение разрозненного контента в виде единой адаптивной сетки (Masonry Layout) с использованием превью
3. Автоматический анализ загруженных изображений и извлечение 3-5 доминантных цветов в палитру.
4. Фильтрация библиотеки по выбранному цвету, набору тегов и конкретному типу файла.
5. Быстрый просмотр полноразмерного изображения или текста по нажатию пробела.
6. Однокнопочное копирование объекта или drag&drop в зону работы.

## 5 - Что не делаем (Out of Scope)

1. Облачная синхронизация. Храним данные только локально (JSON/SQLite).
2. Редактирование. Никакого кропа, фильтров или рисования поверх. Это вьювер/каталогизатор, а не Photoshop.
3. Поддержка анимированных или 3D ассетов с предпросмотром. MVP поддерживает только статику (JPG, PNG, WEBP, SVG).
4. Командный доступ. Приложение только для одного пользователя (Single Player).
5. Версию для телефонов.
